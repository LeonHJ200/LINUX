Terminal y comandos básicos en Linux

La terminal es una herramienta poderosa en Linux que permite ejecutar comandos de manera eficiente. Aprender a usarla es esencial para administrar el sistema, instalar software y automatizar tareas.

1. Navegación por el sistema de archivos

  • pwd → Muestra la ubicación actual.
  • ls → Lista archivos y carpetas en un directorio.
  • cd <ruta> → Cambia al directorio especificado.
  • mkdir <nombre> → Crea una nueva carpeta.
  • rm <archivo> → Borra un archivo.

2. Gestión de archivos y permisos

  • cp <origen> <destino> → Copia un archivo o carpeta.
  • mv <origen> <destino> → Mueve o renombra un archivo.
  • chmod 755 <archivo> → Cambia permisos de acceso.
  • chown usuario:grupo <archivo> → Modifica propietario y grupo.

3. Administración de paquetes

Debian/Ubuntu:

sudo apt update && sudo apt upgrade
sudo apt install <paquete>
sudo apt remove <paquete>

Fedora:

sudo dnf update
sudo dnf install <paquete>
sudo dnf remove <paquete>

Arch Linux:

sudo pacman -Syu
sudo pacman -S <paquete>
sudo pacman -R <paquete>

4. Información del sistema y procesos

  • uname -a → Muestra detalles del kernel.
  • top → Monitorea consumo de CPU y memoria.
  • ps aux → Lista todos los procesos activos.
  • kill <PID> → Termina un proceso específico.

5. Gestión de usuarios y grupos

  • whoami → Muestra el usuario activo.
  • adduser <usuario> → Crea un nuevo usuario.
  • passwd <usuario> → Cambia la contraseña de un usuario.
  • groups <usuario> → Lista los grupos a los que pertenece un usuario.
