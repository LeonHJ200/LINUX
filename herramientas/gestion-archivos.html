Sistemas de gestión de archivos en Linux

1. Comandos básicos para gestión de archivos

Para manejar archivos y directorios desde la terminal:

  • ls → Lista archivos en un directorio.
  • cd <directorio> → Cambia de directorio.
  • mkdir <nombre> → Crea un nuevo directorio.
  • rm <archivo> → Borra un archivo.
  • mv <origen> <destino> → Mueve o renombra un archivo.
  • cp <origen> <destino> → Copia un archivo o directorio.
  • find <ruta> -name "<nombre>" → Busca archivos por nombre.

2. Administradores de archivos gráficos

  • Nautilus: Administrador predeterminado en GNOME, fácil de usar.
  • Dolphin: Potente gestor de KDE Plasma, con opciones avanzadas.
  • Thunar: Ligero y rápido, ideal para XFCE.
  • PCManFM: Alternativa liviana para LXQt.
  • Caja: Administrador usado en el entorno MATE.

3. Montaje y gestión de sistemas de archivos

  • mount /dev/sdX /mnt/punto_de_montaje → Monta un dispositivo.
  • umount /mnt/punto_de_montaje → Desmonta un dispositivo.
  • df -h → Muestra el uso del espacio en disco.
  • du -sh <ruta> → Muestra el tamaño de una carpeta específica.

4. Gestión de permisos y acceso

  • chmod 755 <archivo> → Configura permisos de ejecución.
  • chown usuario:grupo <archivo> → Cambia propietario y grupo de un archivo.
  • ls -l → Muestra permisos detallados de los archivos.

5. Compresión y archivado de archivos

  • tar -cvf archivo.tar carpeta/ → Empaqueta archivos sin comprimir.
  • tar -xvf archivo.tar → Extrae un paquete .tar.
  • zip archivo.zip carpeta/ → Comprime en formato ZIP.
  • unzip archivo.zip → Extrae un archivo ZIP.
  • gzip archivo.txt → Comprime un archivo en formato .gz.

6. Sincronización y copias de seguridad

Para mantener archivos seguros y sincronizados entre dispositivos:

  • rsync: Sincroniza archivos entre directorios o sistemas.
  • rsync -av --progress origen/ destino/
  • Timeshift: Herramienta gráfica para copias de seguridad del sistema.
  • BorgBackup: Solución eficiente para respaldos incrementales y cifrados.
